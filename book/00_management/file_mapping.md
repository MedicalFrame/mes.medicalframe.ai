## 빌드 관련 메모

- 현재 합본 출력 파일은 `book_manuscript/book.md`다.
- DOCX 참조 스타일 문서는 `user/의공모.docx`다.
- 실제 빌드 순서와 파일 목록은 `00_management/scripts/build_book.py`가 최종 기준이다.
- 제작 노트 파일은 `PARTS` 목록에 직접 들어가 있지 않고, `build_book.py` 마지막 단계에서 별도 append 된다.
- 프롤로그는 원본 파일을 보존하고, 현재 빌드에는 축약판 파일을 연결한다.
